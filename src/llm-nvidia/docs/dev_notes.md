⚠️ 고려해야 할 점
학습 곡선: LangChain은 진입 장벽이 낮아 빠르게 시작 가능하지만, LangGraph는 구조적 사고가 필요해 초기 진입은 어렵습니다.

확장성: 단순한 프로젝트를 LangChain으로 시작했다가, 복잡성이 커지면 LangGraph로 옮기는 전략이 흔히 쓰입니다.

메모리: LangChain은 기본적으로 메모리 관리 기능을 제공하지만, LangGraph는 상태 관리 자체를 개발자가 직접 설계해야 합니다.
